# Cybersecurity Journey Documentation

## CPU Consumption
CPU consumption refers to the amount of processing power used by processes and services running on a computer. It is usually expressed as a percentage of the total CPU capacity. High CPU consumption can slow down system performance and may indicate that a process or service is using excessive resources.

## What is a Process?
A process is an instance of a program that is being executed. It contains the program code and its current activity. Processes are essential for running software applications on a computer and can communicate with one another and with the operating system.

### Key Features of a Process:
- **Memory Allocation**: Each process has its own memory space.
- **Execution**: Processes run independently and can be managed by the operating system.

## What is a Service?
A service is a background process that operates without user intervention. It is typically started when the operating system boots and runs in the background, providing essential functions for other applications. Services can also be triggered by specific events.

### Key Features of a Service:
- **Runs in the Background**: Services operate without a user interface.
- **Provides Functionality**: They perform tasks on behalf of other applications or users, such as managing network connections, running scheduled tasks, and more.

### Conclusion
Understanding CPU consumption along with processes and services is crucial for managing system performance and troubleshooting issues effectively.
